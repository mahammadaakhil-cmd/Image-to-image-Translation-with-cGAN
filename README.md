# Image-to-Image Translation with cGAN (Pix2Pix)

## 📌 Overview

This project implements an Image-to-Image Translation model using a Conditional Generative Adversarial Network (cGAN) based on the Pix2Pix architecture. The model learns to translate an input image into a corresponding output image using paired training data.

This project was completed as part of the **Prodigy InfoTech AI Internship – Task 04**.

## 🎯 Objective

Develop a Pix2Pix model that performs image-to-image translation by learning the mapping between paired input and target images.

## 🛠️ Technologies Used

- Python
- TensorFlow
- NumPy
- Matplotlib
- Google Colab

## 📂 Dataset

The project uses the **Facades Dataset**, which contains paired images of building labels and real building photographs.

Dataset Source:
http://efrosgans.eecs.berkeley.edu/pix2pix/datasets/

## 🚀 Features

- Image-to-Image Translation using Pix2Pix
- Conditional GAN (cGAN) architecture
- Generator based on U-Net
- PatchGAN Discriminator
- Training and inference on paired image datasets

## ▶️ Running the Project

1. Open the notebook in Google Colab.
2. Install dependencies.
3. Download and extract the Facades dataset.
4. Train or load the Pix2Pix model.
5. Test the model with sample images.
6. View the generated output images.

## 📊 Expected Output

Input:
- Building layout / segmentation map

Output:
- Realistic building image generated by the Pix2Pix model

The model learns to convert semantic building layouts into realistic photographs.
